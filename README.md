# ENS Public Goods Working Group - Term 6 (2025) Dashboard

This repository contains the comprehensive dashboard showcasing the achievements and impact of the ENS Public Goods Working Group during Term 6 (2025).

## Overview

The dashboard provides:
- **Strategic Grants Timeline**: Interactive timeline of all 6 strategic grants awarded throughout Term 6
- **Builder Grants Overview**: Statistics and examples from 46 builder grants distributed
- **Impact Stories**: Detailed case studies showing real-world outcomes (Revoke.cash, Vyper, Fabric)
- **Data Visualizations**: Interactive charts showing grant distribution, funding breakdown, and geographic reach
- **Transparency**: Links to all forum posts, Dune dashboard, and source materials

## Key Metrics

### Strategic Grants
- **$425,000 USDC** committed across 6 projects
- Co-funded with Ethereum Foundation
- Focus: Developer tooling, compiler diversity, L2 infrastructure, policy advocacy

### Builder Grants
- **57.75 ETH + 10,000 USDC** distributed
- **46 grants** awarded (43 small, 3 large)
- **278 proposals** submitted
- **26+ unique grantees** across 6 continents

## Technology Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- shadcn/ui components
- Recharts for data visualization
- Wouter for routing

## Getting Started

### Prerequisites
- Node.js 22.x
- pnpm

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Deployment

This site can be deployed to:
- GitHub Pages
- Vercel
- Netlify
- Any static hosting service
